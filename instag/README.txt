1.install python (v-3.11).

2.Open command prompt, navigate to instag folder install required python libraries by executing this command "pip install -r requirements.txt".

3.Install instagram from microsoft store.

4.Create a shortcut for instagram in your desktop.

5.Turn on dark theme in instagram.

5.Copy the shortcut path(right click the shortcut, click "copy as path"), execute the setup.py and paste the path.

6.execute the main.py.


Features And Commamds():-

1.Opening and closing the instagram (open instagram)

2.Timer (set timer for 1 minute)

3.Voice Feedback (a.turn on => turn on voice feedback __ b.turn off => turn off voice feedback)

4.display the user data (show dashboard,show reels count,show usage time   or  replace show by display) 
note:- "close the figure window to continue listening"

5.Open different pages of instagram(open reels, open home, open meassges,open notifications, open profile, open settings)

6.Mute and unmute the videos (mute,unmute)

7.Sharing the videos (share to "id_name")

8.Save and unsave the videos (save,unsave)

9.Post comments (comment "your comment", open comments, close comments)

10.follow and unfollow accounts in reels page (follow this id, unfollow this id)

11.Change the reels(next reel,previous reel)

12.Opening the unread messages (open unread messages)

13.Sending messages (send "your message" to "id")

14.Counting the unread messages (how many unread messages i have)
